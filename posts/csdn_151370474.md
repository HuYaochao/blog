### 1.1 变量：给数据起个名字

#### > 什么是变量

想象一下，你有一个标签纸，可以贴在不同的盒子上。今天这个标签贴在装苹果的盒子上，明天可以撕下来贴在装橘子的盒子上。在Python中，\*\*变量（Variable）\*\*就像这个标签，用来给数据起名字，方便我们使用。

```
# 给数字起名字
age = 25
print(age)  # 输出：25

# 给文字起名字
name = "小明"
print(name)  # 输出：小明

# 变量可以改变存储的内容
age = 26  # 现在age存储的是26
print(age)  # 输出：26
```

![在这里插入图片描述](https://i-blog.csdnimg.cn/direct/66b94a1088c441bcbfc64574cf11aefd.png)

#### > 变量命名规则

就像给孩子起名字有一些约定俗成的规则一样，Python变量命名也有规则：

```
# 正确的变量名
student_name = "张三"      # 用下划线连接多个单词
age2023 = 20              # 可以包含数字，但不能以数字开头
user_email = "abc@qq.com" # 描述性的名字更容易理解

# 错误的变量名示例（这些会报错）
# 2age = 20        # 不能以数字开头
# user-name = ""   # 不能包含横线
# for = 10         # 不能使用Python关键字
```

**记忆技巧**：变量名要像路标一样清楚，让别人（包括未来的你）一看就知道里面装的是什么。

### 1.2 数据类型：不同种类的数据

#### > 数字类型

Python中有两种常见的数字类型，就像我们生活中区分整数和小数一样：

```
# 整数（没有小数点的数字）
student_count = 30        # 学生人数
book_pages = 256          # 书的页数

# 小数（带小数点的数字）
height = 1.75             # 身高
temperature = 36.5        # 体温
price = 29.99             # 商品价格

print(student_count + 5)  # 整数计算：35
print(height * 2)         # 小数计算：3.5
```

#### > 文字类型

在Python中，文字被称为**字符串（String）**，需要用引号包起来：

```
# 用双引号包围文字
message = "欢迎来到Python世界"
book_title = "Python入门教程"

# 用单引号也可以
author = '张老师'
city = '北京'

# 字符串可以相加（连接）
full_name = "张" + "三丰"
print(full_name)  # 输出：张三丰

greeting = "你好，" + "小明"
print(greeting)   # 输出：你好，小明
```

#### > 真假类型

有些情况下，我们只需要表示"是"或"不是"，Python提供了**布尔类型（Boolean）**：

```
is_student = True         # 是学生
has_homework = False      # 没有作业
is_raining = True         # 正在下雨

print(is_student)         # 输出：True
print(has_homework)       # 输出：False
```

### 1.3 简单的条件判断

#### > if语句：如果…那么…

生活中我们经常做判断：“如果下雨，那么带伞”。Python的条件判断也是这个逻辑：

```
# 基本的条件判断
weather = "下雨"

if weather == "下雨":
    print("记得带伞")
else:
    print("可以不带伞")
```

#### > 数字比较

```
score = 85

if score >= 90:
    print("成绩优秀")
elif score >= 70:  # elif意思是"否则如果"
    print("成绩良好")
else:
    print("需要努力")
```

#### > 多个条件的组合

```
age = 20
has_id = True

# and表示"并且"，两个条件都要满足
if age >= 18 and has_id == True:
    print("可以进入网吧")
else:
    print("不能进入网吧")

# or表示"或者"，满足其中一个条件即可
day = "周末"
weather = "晴天"

if day == "周末" or weather == "晴天":
    print("适合出去玩")
```

### 1.4 循环：重复做同样的事

#### > for循环：已知要重复多少次

有时候我们需要重复做同样的事情，比如打印1到5的数字：

```
# 打印1到5
for i in range(1, 6):  # range(1, 6)生成1, 2, 3, 4, 5
    print(f"这是第{i}个数字")

# 输出：
# 这是第1个数字
# 这是第2个数字
# 这是第3个数字
# 这是第4个数字
# 这是第5个数字
```

```
# 处理一组名字
names = ["小明", "小红", "小刚"]

for name in names:
    print(f"你好，{name}！")

# 输出：
# 你好，小明！
# 你好，小红！
# 你好，小刚！
```

#### > while循环：满足条件就继续

有时候我们不知道要重复多少次，只知道什么时候停止：

```
# 数数游戏：从1数到10
count = 1

while count <= 10:
    print(f"现在数到了{count}")
    count = count + 1  # 每次循环后，count增加1

print("数数结束！")
```

### 1.5 列表：管理一组相关的数据

#### > 什么是列表

**列表（List）** 就像一个购物清单，可以存放多个相关的东西：

```
# 创建购物清单
shopping_list = ["苹果", "牛奶", "面包", "鸡蛋"]
print(shopping_list)

# 创建分数列表
scores = [85, 92, 78, 96, 88]
print(scores)
```

#### > 列表的基本操作

```
fruits = ["苹果", "香蕉", "橘子"]

# 看看列表里有什么
print(fruits[0])    # 第1个水果：苹果（注意：Python从0开始数）
print(fruits[1])    # 第2个水果：香蕉
print(fruits[-1])   # 最后一个水果：橘子

# 添加新水果
fruits.append("葡萄")
print(fruits)       # ['苹果', '香蕉', '橘子', '葡萄']

# 看看有多少个水果
print(len(fruits))  # 4个水果
```

#### > 列表和循环的配合

```
subjects = ["语文", "数学", "英语", "物理"]

print("今天要学的科目：")
for subject in subjects:
    print(f"- {subject}")
```

### 1.6 实战项目：简单的成绩管理器

现在让我们把学到的知识组合起来，做一个简单的程序：

```
def grade_manager():
    """简单的成绩管理程序"""
    
    print("欢迎使用成绩管理器！")
    print("=" * 30)
    
    # 存储学生姓名和成绩
    student_names = []
    student_scores = []
    
    # 输入学生信息
    student_count = 3  # 先处理3个学生
    
    for i in range(student_count):
        print(f"\n请输入第{i+1}个学生的信息：")
        
        # 输入姓名
        name = input("学生姓名：")
        student_names.append(name)
        
        # 输入成绩
        score_text = input("学生成绩：")
        score = int(score_text)  # 将文字转换为数字
        student_scores.append(score)
    
    # 显示所有学生信息
    print("\n" + "=" * 30)
    print("学生成绩汇总：")
    print("=" * 30)
    
    total_score = 0
    for i in range(len(student_names)):
        name = student_names[i]
        score = student_scores[i]
        total_score = total_score + score
        
        # 判断成绩等级
        if score >= 90:
            grade = "优秀"
        elif score >= 80:
            grade = "良好"
        elif score >= 60:
            grade = "及格"
        else:
            grade = "不及格"
        
        print(f"{name}：{score}分（{grade}）")
    
    # 计算平均分
    average_score = total_score / len(student_names)
    print(f"\n班级平均分：{average_score:.1f}分")
    
    # 找出最高分
    highest_score = max(student_scores)
    highest_index = student_scores.index(highest_score)
    best_student = student_names[highest_index]
    
    print(f"最高分：{best_student}（{highest_score}分）")
    
    print("\n成绩统计完成！")

# 运行程序（注意：在实际环境中才能输入）
# grade_manager()

# 为了演示效果，我们用预设数据运行一个简化版本
def demo_grade_manager():
    """演示版本的成绩管理器"""
    
    print("成绩管理器演示")
    print("=" * 20)
    
    # 预设的学生数据
    names = ["小明", "小红", "小刚"]
    scores = [85, 92, 78]
    
    print("学生成绩：")
    total = 0
    
    for i in range(len(names)):
        name = names[i]
        score = scores[i]
        total = total + score
        
        if score >= 90:
            level = "优秀"
        elif score >= 80:
            level = "良好"
        else:
            level = "及格"
            
        print(f"{name}: {score}分 ({level})")
    
    average = total / len(names)
    print(f"\n平均分: {average:.1f}分")

# 运行演示
demo_grade_manager()
```

### 附录：专业术语表

> **变量（Variable）**：用来存储和引用数据的名字，就像给数据贴上标签

> **字符串（String）**：文字数据类型，需要用引号包围

> **布尔类型（Boolean）**：只有True（真）和False（假）两个值的数据类型

> **列表（List）**：用来存储多个相关数据的容器，就像购物清单

> **循环（Loop）**：让程序重复执行相同代码的结构

> **条件判断（Conditional Statement）**：让程序根据不同情况执行不同代码的结构
