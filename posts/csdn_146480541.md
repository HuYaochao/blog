## 一、监督学习模型综合对比表

下面的表格详细列出常见的监督学习模型，从经典的线性模型、判别分析，到支持向量机、树模型、朴素贝叶斯、最近邻、神经网络及其他较为前沿的方法，涵盖其主要用途、优势、缺点、适用场景以及在 Python 中常用的工具。（神经网络表格单独说）

| 模型类别 | 模型名称（中文/英文） | 主要用途 | 优点 | 缺点 | 适用场景 | 常用 Python 库/工具 |
| --- | --- | --- | --- | --- | --- | --- |
| **线性模型** | 普通最小二乘回归 / Ordinary Least Squares | 连续变量预测、趋势分析 | 模型简单、解释性强、计算高效 | 对异常值敏感、仅适用于线性关系 | 销售预测、经济趋势、温度预测 | scikit-learn (LinearRegression) |
|  | 岭回归 / Ridge Regression | 连续变量预测，解决多重共线性 | 稳定性好、抑制过拟合 | 不进行特征选择 | 经济数据、环境监测 | scikit-learn (Ridge) |
|  | LASSO 回归 / LASSO | 连续变量预测与自动特征选择 | 自动变量筛选、产生稀疏解 | 可能过度稀疏、参数调优复杂 | 信贷评分、市场预测 | scikit-learn (Lasso, LassoCV) |
|  | 弹性网 / Elastic Net | 连续变量预测与特征选择 | 综合 LASSO 和岭回归优势、灵活调节 | 需要调节两个正则化参数 | 高维数据建模 | scikit-learn (ElasticNet) |
|  | 多任务 LASSO / Multi-task LASSO | 同时预测多个相关目标变量 | 共享信息，提升多个任务的预测精度 | 模型复杂性增加 | 多输出问题、联合预测 | scikit-learn (MultiTaskLasso) |
|  | 最小角回归（LARS Lasso） / Least Angle Regression (LARS) | 高维数据下高效特征选择 | 计算高效、适合高维数据 | 理论复杂、仅适用于线性关系 | 基因数据、文本特征 | scikit-learn (Lars, LassoLars) |
|  | 正态贝叶斯回归 / Bayesian Regression | 连续变量预测 | 提供不确定性估计、能整合先验知识 | 计算复杂度较高、对先验假设敏感 | 金融风险评估、医学预测 | PyMC3, scikit-learn (BayesianRidge) |
| **判别分析** | 线性判别分析 / Linear Discriminant Analysis (LDA) | 分类与降维 | 强化类别分离、降维效果好 | 仅适用于正态分布数据 | 医疗诊断、客户分类 | scikit-learn (LinearDiscriminantAnalysis) |
|  | 二次判别分析 / Quadratic Discriminant Analysis (QDA) | 分类 | 可捕捉类别间非线性差异 | 对样本量要求较高、估计协方差矩阵不稳定 | 信贷风险、复杂分类问题 | scikit-learn (QuadraticDiscriminantAnalysis) |
| **支持向量机** | 支持向量机 / Support Vector Machines (SVM) | 分类、回归、异常检测 | 适用于高维数据、决策边界清晰 | 参数调优复杂、计算资源需求高 | 图像识别、文本分类、风控 | scikit-learn (SVC, SVR) |
| **最近邻** | k-近邻 / k-Nearest Neighbors (kNN) | 分类与回归 | 模型简单、无需训练过程、直观 | 计算量大、对噪声敏感 | 推荐系统、模式识别、医疗诊断 | scikit-learn (KNeighborsClassifier, KNeighborsRegressor) |
| **高斯过程** | 高斯过程回归/分类 / Gaussian Processes (GP) | 回归与分类 | 非参数模型，能提供预测不确定性 | 计算复杂度高，不适用于大数据 | 小规模数据预测、风险评估 | scikit-learn (GaussianProcessRegressor, GaussianProcessClassifier) |
| **朴素贝叶斯** | 朴素贝叶斯 / Naive Bayes | 分类 | 计算速度快、适用于高维数据 | 条件独立假设过于简化，可能导致精度不足 | 垃圾邮件检测、情感分析、文档分类 | scikit-learn (GaussianNB, MultinomialNB, BernoulliNB) |
| **决策树** | 决策树 / Decision Trees | 分类与回归 | 直观易懂、可解释性强、处理混合数据 | 容易过拟合，单棵树稳定性差 | 信贷风险评估、客户分类、医学诊断 | scikit-learn (DecisionTreeClassifier, DecisionTreeRegressor) |
| **集成方法** | 随机森林 / Random Forests | 分类与回归 | 抗噪声、降低过拟合、提供特征重要性 | 模型解释性较差、计算资源需求高 | 风险管理、市场预测、复杂分类问题 | scikit-learn (RandomForestClassifier, RandomForestRegressor), XGBoost, LightGBM |
|  | 梯度提升树 / Gradient Boosting | 分类与回归 | 高精度、处理非线性关系优秀 | 调参复杂、训练时间长 | 信贷评分、推荐系统、复杂预测模型 | scikit-learn (GradientBoostingClassifier, GradientBoostingRegressor), XGBoost |
| **其他方法** | 支持向量回归 / Support Vector Regression (SVR) | 回归 | 处理非线性关系，鲁棒性好 | 计算复杂度高，参数调优复杂 | 股票价格预测、需求预测 | scikit-learn (SVR) |
|  | 基于树模型特征筛选 / Tree-based Feature Selection | 特征筛选与降维 | 能捕捉非线性关系，提供直观特征重要性评分 | 依赖于模型表现，结果解释性较差 | 数据预处理、模型简化 | scikit-learn (RandomForestRegressor), XGBoost |
|  | 线性判别分析 / Linear Discriminant Analysis (LDA) | 分类降维与特征提取 | 强化类别分离，降维效果好 | 仅适用于有标签数据，假设正态分布 | 客户分群、医疗诊断、模式识别 | scikit-learn (LinearDiscriminantAnalysis) |
|  | t-SNE 与 UMAP / t-SNE & UMAP | 非线性降维与数据可视化 | 能揭示高维数据簇结构，效果优秀 | 计算复杂度高，参数敏感，仅适用于数据可视化 | 数据探索、模式识别、可视化 | scikit-learn (t-SNE), umap-learn |
|  | 线性判别分析（分类版）/ Linear Discriminant Analysis (for classification) | 分类降维与特征提取 | 能最大化类别间差异，提高模型区分能力 | 仅适用于有监督数据，数据正态性要求较高 | 文本分类、客户分类、医学诊断 | scikit-learn (LinearDiscriminantAnalysis) |
|  | 贝叶斯回归 / Bayesian Regression | 连续变量预测及不确定性估计 | 能结合先验信息，提供预测不确定性估计 | 计算复杂，模型假设较多 | 风险评估、金融预测、医学预测 | scikit-learn (BayesianRidge), PyMC3 |

## 二、深度学习模型综合对比表

下面的表格详细列出了深度学习中常见的模型类型，从基础前馈网络到的 Transformer 模型，涵盖各模型的基本描述、主要优缺点、适用场景及在 Python 中常用的工具。这份表格旨在帮助您全面理解深度学习模型的分类及其应用，为项目选型和研究提供参考。

| 模型类别 | 模型名称（中文/英文） | 描述 | 优点 | 缺点 | 适用场景 | 常用 Python 库/工具 |
| --- | --- | --- | --- | --- | --- | --- |
| **前馈神经网络** | 多层感知机 / Multi-layer Perceptron (MLP) | 基础前馈神经网络，由输入层、隐藏层和输出层构成，用于非线性建模 | 结构简单，灵活可扩展；适用于多种任务 | 需要大量数据训练；对超参数调优敏感 | 图像识别（简单任务）、回归、分类 | TensorFlow, Keras, PyTorch, scikit-learn (MLPClassifier, MLPRegressor) |
| **卷积神经网络** | 卷积神经网络 / Convolutional Neural Networks (CNN) | 利用卷积层提取局部特征，通常用于处理图像、视频等二维数据 | 自动提取局部特征，参数共享降低复杂度；在图像领域表现卓越 | 对硬件要求较高；需要大量训练数据；设计较复杂 | 图像分类、物体检测、医学影像、视频分析 | TensorFlow, Keras, PyTorch |
| **循环神经网络** | 循环神经网络 / Recurrent Neural Networks (RNN) | 用于处理序列数据，具有内部状态，能捕捉时间依赖性 | 适合时间序列和序列数据处理，能够捕捉数据的时序特征 | 容易出现梯度消失或爆炸问题；长序列处理效果有限 | 自然语言处理、语音识别、时间序列预测 | TensorFlow, Keras, PyTorch |
|  | 长短时记忆网络 / Long Short-Term Memory (LSTM) | RNN 的改进版本，通过门控机制有效捕捉长程依赖 | 能较好地缓解梯度消失问题，捕捉长程依赖 | 计算复杂度高、模型结构庞大，训练时间长 | 机器翻译、文本生成、语音识别、时间序列预测 | TensorFlow, Keras, PyTorch |
|  | 门控循环单元 / Gated Recurrent Unit (GRU) | 类似 LSTM，但结构更简化，门控机制较少参数，运行更快 | 结构更简洁，计算效率高，性能接近 LSTM | 表达能力可能略逊于 LSTM | 短文本处理、实时系统、语音识别 | TensorFlow, Keras, PyTorch |
| **自注意力与 Transformer** | Transformer | 基于自注意力机制，完全抛弃 RNN 架构，支持并行化训练 | 高效捕捉长程依赖，支持大规模并行计算，适用于大规模预训练模型 | 模型结构复杂，对计算资源要求高，需要大量数据 | 机器翻译、文本生成、问答系统、预训练语言模型 | TensorFlow, Keras, PyTorch (Hugging Face Transformers) |
| **生成模型** | 生成对抗网络 / Generative Adversarial Networks (GAN) | 由生成器与判别器组成，通过对抗训练生成逼真数据 | 能生成高质量图像和其他数据，应用广泛，如图像生成与数据增强 | 训练不稳定，模式崩溃风险较高，调参困难 | 图像生成、数据增强、风格迁移、视频生成 | TensorFlow, Keras, PyTorch (GAN 框架) |
| **自编码器** | 自编码器 / Autoencoder | 通过编码器将数据压缩到低维，再通过解码器重构原始数据，用于降噪与特征提取 | 可用于无监督特征提取和降噪，结构简单，易于扩展 | 重构误差可能较高，需要精心设计网络结构 | 图像降噪、特征提取、异常检测 | TensorFlow, Keras, PyTorch |
| **图神经网络** | 图神经网络 / Graph Neural Networks (GNN) | 处理图结构数据，通过消息传递学习节点和边的表示 | 能处理复杂网络数据，捕捉节点间关系，适用于非欧几里得数据 | 算法和模型设计复杂，调参困难 | 社交网络分析、推荐系统、分子结构预测 | PyTorch Geometric, DGL (Deep Graph Library) |
| **深度置信网络** | 深度置信网络 / Deep Belief Networks (DBN) | 多层生成模型，通过堆叠受限玻尔兹曼机进行无监督预训练，再进行微调 | 能初始化深度神经网络参数，适用于无监督学习，较早期的深度学习模型 | 训练复杂性高，现已被更先进的神经网络架构取代 | 早期图像识别、语音处理、数据预训练 | Theano, TensorFlow (较少使用) |
