### 1.1 冯·诺依曼架构：现代计算机的奠基石

#### > 什么是冯·诺依曼架构

想象一下，你正在按照菜谱做菜。菜谱（程序）告诉你每一步该做什么，你（处理器）按照菜谱的指示，从冰箱（内存）取出食材（数据），用厨具（运算器）进行加工，最后把成品（结果）装盘（输出）。这就是 **冯·诺依曼架构（Von Neumann Architecture）** 的基本思想。

**冯·诺依曼架构（Von Neumann Architecture）** 的核心创新在于"**存储程序概念（Stored Program Concept）**"：程序指令和数据都存储在同一个内存空间中，处理器可以像读取数据一样读取和修改程序指令。这个看似简单的概念，却是现代所有计算设备的基础。

让我们通过一个具体例子理解这个过程：

```
假设要执行：c = a + b （其中 a=5, b=3）

1. 取指令（Fetch）：
   - CPU从内存地址0x1000读取指令"ADD R1, R2, R3"
   - 程序计数器PC更新为下一条指令地址

2. 译码（Decode）：
   - 控制单元分析指令：这是一个加法操作
   - 识别操作数：R1和R2寄存器的值，结果存入R3

3. 执行（Execute）：
   - 算术逻辑单元ALU执行：5 + 3 = 8
   - 将结果8写入寄存器R3

4. 存储结果（Store）：
   - 如需要，将R3的值写回内存
   - 准备执行下一条指令
```

#### > CPU：复杂指令的执行专家

### 1.4 应用类型识别与硬件匹配

#### > 计算密集型应用特征

**计算密集型（Compute-Intensive）** 应用的核心特征是需要大量的数学运算，相对较少的内存访问：

**典型应用场景**：

- **机器学习训练**：神经网络前向传播和反向传播
- **科学计算**：流体动力学模拟、气象预测
- **密码学运算**：RSA加密、区块链挖矿
- **图像处理**：卷积、滤波、变换

**性能特征指标**：

```
计算密集度 = 浮点运算次数 / 内存访问次数
计算密集型应用通常：Compute/Memory Ratio > 10:1
```

**硬件需求分析**：

- **处理器**：高核心数、高频率CPU或GPU
- **内存**：容量适中，更注重带宽而非延迟
- **存储**：对存储I/O要求相对较低
- **网络**：单机计算为主，网络要求不高

#### > 内存密集型应用特征

**内存密集型（Memory-Intensive）** 应用的特征是需要频繁访问大量数据，计算相对简单：

**典型应用场景**：

- **数据库系统**：大表连接、索引扫描
- **大数据分析**：数据聚合、排序、过滤
- **图计算**：社交网络分析、推荐系统
- **内存数据库**：Redis、Memcached等

**性能特征指标**：

```
内存访问密度 = 内存访问次数 / 浮点运算次数  
内存密集型应用通常：Memory/Compute Ratio > 10:1
```

**硬件需求分析**：

- **内存**：大容量（128GB+）、高带宽、低延迟
- **处理器**：注重内存控制器性能，核心数适中
- **存储**：高IOPS、低延迟的NVMe SSD
- **网络**：分布式系统需要高带宽网络

#### > 应用类型对比与硬件配置建议

| 应用类型 | 计算密集型 | 内存密集型 | 混合型 |
| --- | --- | --- | --- |
| **主要瓶颈** | 运算单元数量和频率 | 内存容量和带宽 | 均衡配置 |
| **CPU配置** | 高核心数（32-64核+） | 大缓存、多内存通道 | 中等核心数（16-32核） |
| **GPU加速** | 强烈推荐（CUDA/OpenCL） | 一般不需要 | 视具体算法而定 |
| **内存配置** | 64-128GB，高带宽 | 256GB-2TB+，低延迟 | 128-512GB，均衡 |
| **存储需求** | 中等，主要存模型参数 | 高IOPS，大容量缓存 | 高性能NVMe阵列 |
| **网络需求** | 模型并行时需要高带宽 | 数据分片时需要高带宽 | 根据架构设计 |

#### > 核心监控指标定义

**CPU性能指标**：

- **利用率（Utilization）**：各核心的使用百分比
- **负载平均值（Load Average）**：1、5、15分钟平均负载
- **上下文切换（Context Switches）**：进程切换频率
- **中断处理（Interrupts）**：硬件中断和软中断数量

**内存性能指标**：

- **使用率（Memory Usage）**：已用内存/总内存
- **缓存效率（Cache Hit Ratio）**：L1、L2、L3缓存命中率
- **页面错误（Page Faults）**：主缺页和次缺页频率
- **内存带宽（Memory Bandwidth）**：读写带宽利用率

**GPU性能指标**：

- **GPU利用率（GPU Utilization）**：计算单元使用率
- **显存使用（Memory Usage）**：显存占用量和带宽
- **温度功耗（Temperature/Power）**：热管理和功耗分析
- **CUDA核心利用率（CUDA Core Utilization）**：并行度分析

import psutil
import shutil
import time
import json
import subprocess
import platform
import os
import string
from datetime import datetime
from collections import defaultdict

class HardwareMonitor:
    def __init__(self):
        self.data_buffer = defaultdict(list)
        self.monitoring = False
        
    def safe_round(self, value, decimals=2):
        try:
            return round(float(value), decimals)
        except:
            return 0.0
            
    def safe_int(self, value):
        try:
            return int(float(value))
        except:
            return 0
    
    def get_cpu_stats(self):
        """CPU监控"""
        try:
            try:
                load_avg = psutil.getloadavg()
            except:
                load_avg = (0.0, 0.0, 0.0)
            
            cpu_percent = psutil.cpu_percent(interval=1, percpu=True)
            cpu_stats = psutil.cpu_stats()
            
            return {
                'timestamp': datetime.now().isoformat(),
                'cpu_percent': cpu_percent,
                'cpu_avg': self.safe_round(sum(cpu_percent) / len(cpu_percent)),
                'cpu_max': self.safe_round(max(cpu_percent)),
                'load_avg': load_avg,
                'logical_cores': psutil.cpu_count(),
                'physical_cores': psutil.cpu_count(logical=False),
                'context_switches': cpu_stats.ctx_switches
            }
        except Exception as e:
            print("CPU监控错误: " + str(e))
            return None
    
    def get_memory_stats(self):
        """内存监控"""
        try:
            mem = psutil.virtual_memory()
            swap = psutil.swap_memory()
            
            return {
                'timestamp': datetime.now().isoformat(),
                'total_gb': self.safe_round(mem.total / (1024**3)),
                'used_gb': self.safe_round(mem.used / (1024**3)),
                'available_gb': self.safe_round(mem.available / (1024**3)),
                'percent': self.safe_round(mem.percent),
                'swap_total_gb': self.safe_round(swap.total / (1024**3)),
                'swap_used_gb': self.safe_round(swap.used / (1024**3)),
                'swap_percent': self.safe_round(swap.percent)
            }
        except Exception as e:
            print("内存监控错误: " + str(e))
            return None
    
    def get_disk_stats(self):
        """磁盘监控 - 使用shutil修复版"""
        disk_data = {}
        io_data = {}
        
        # 磁盘使用情况 - 使用shutil.disk_usage()
        try:
            if platform.system() == 'Windows':
                for drive_letter in string.ascii_uppercase:
                    drive_path = drive_letter + ':\\'
                    if os.path.exists(drive_path):
                        try:
                            usage = shutil.disk_usage(drive_path)
                            total, used, free = usage.total, usage.total - usage.free, usage.free
                            
                            if total > 1024 * 1024 * 100:  # 至少100MB
                                disk_key = 'disk_' + drive_letter
                                disk_data[disk_key] = {
                                    'device': drive_letter + ':',
                                    'mountpoint': drive_path,
                                    'fstype': 'NTFS',
                                    'total_gb': self.safe_round(total / (1024**3)),
                                    'used_gb': self.safe_round(used / (1024**3)),
                                    'free_gb': self.safe_round(free / (1024**3)),
                                    'percent': self.safe_round(used / max(total, 1) * 100)
                                }
                        except:
                            continue
            else:
                # Linux/Unix系统
                partitions = psutil.disk_partitions()
                for i, partition in enumerate(partitions[:4]):
                    if (partition.fstype and 
                        not partition.mountpoint.startswith(('/dev', '/proc', '/sys', '/run'))):
                        try:
                            usage = shutil.disk_usage(partition.mountpoint)
                            total, used, free = usage.total, usage.total - usage.free, usage.free
                            
                            if total > 0:
                                disk_key = 'disk_' + str(i)
                                disk_data[disk_key] = {
                                    'device': str(partition.device),
                                    'mountpoint': str(partition.mountpoint),
                                    'fstype': str(partition.fstype),
                                    'total_gb': self.safe_round(total / (1024**3)),
                                    'used_gb': self.safe_round(used / (1024**3)),
                                    'free_gb': self.safe_round(free / (1024**3)),
                                    'percent': self.safe_round(used / max(total, 1) * 100)
                                }
                        except:
                            continue
        except Exception as e:
            print("磁盘使用监控错误: " + str(e))
        
        # IO统计
        try:
            disk_io = psutil.disk_io_counters(perdisk=True)
            if disk_io:
                for device, io in list(disk_io.items())[:5]:
                    try:
                        device_key = str(device).replace('\\', '').replace(':', '').replace(' ', '_')
                        io_data[device_key] = {
                            'read_mb': self.safe_round(io.read_bytes / (1024**2)),
                            'write_mb': self.safe_round(io.write_bytes / (1024**2)),
                            'read_count': self.safe_int(io.read_count),
                            'write_count': self.safe_int(io.write_count)
                        }
                    except:
                        continue
        except Exception as e:
            print("磁盘IO监控错误: " + str(e))
        
        return {
            'timestamp': datetime.now().isoformat(),
            'usage': disk_data,
            'io': io_data
        }
    
    def get_gpu_stats(self):
        """GPU监控"""
        gpu_data = []
        smi_paths = [
            'nvidia-smi',
            'C:\\Program Files\\NVIDIA Corporation\\NVSMI\\nvidia-smi.exe',
            '/usr/bin/nvidia-smi'
        ]
        
        for smi_path in smi_paths:
            try:
                cmd = [smi_path, '--query-gpu=index,name,temperature.gpu,utilization.gpu,utilization.memory,memory.total,memory.used,power.draw', '--format=csv,noheader,nounits']
                creation_flags = subprocess.CREATE_NO_WINDOW if platform.system() == 'Windows' else 0
                result = subprocess.run(cmd, capture_output=True, text=True, timeout=10, creationflags=creation_flags)
                
                if result.returncode == 0 and result.stdout.strip():
                    for line in result.stdout.strip().split('\n'):
                        if line and ',' in line:
                            parts = [p.strip() for p in line.split(',')]
                            if len(parts) >= 7:
                                gpu_info = {
                                    'gpu_id': self.safe_int(parts[0]),
                                    'name': str(parts[1]),
                                    'temperature': self.safe_round(parts[2] if parts[2] != 'N/A' else 0),
                                    'gpu_util': self.safe_round(parts[3] if parts[3] != 'N/A' else 0),
                                    'mem_util': self.safe_round(parts[4] if parts[4] != 'N/A' else 0),
                                    'mem_total_mb': self.safe_int(parts[5] if parts[5] != 'N/A' else 0),
                                    'mem_used_mb': self.safe_int(parts[6] if parts[6] != 'N/A' else 0),
                                    'power_draw': self.safe_round(parts[7] if len(parts) > 7 and parts[7] != 'N/A' else 0)
                                }
                                gpu_info['mem_total_gb'] = self.safe_round(gpu_info['mem_total_mb'] / 1024)
                                gpu_info['mem_used_gb'] = self.safe_round(gpu_info['mem_used_mb'] / 1024)
                                gpu_data.append(gpu_info)
                    break
            except:
                continue
        
        return {
            'timestamp': datetime.now().isoformat(),
            'gpu_count': len(gpu_data),
            'gpu_data': gpu_data
        }
    
    def get_network_stats(self):
        """网络监控"""
        try:
            net_io = psutil.net_io_counters(pernic=True)
            connections = psutil.net_connections()
            
            interface_data = {}
            total_sent = total_recv = 0
            
            for name, io in net_io.items():
                if (not name.lower().startswith(('lo', 'loopback', 'isatap', 'teredo')) and 
                    io.bytes_sent > 0 and len(interface_data) < 5):
                    
                    interface_data[str(name)] = {
                        'sent_mb': self.safe_round(io.bytes_sent / (1024**2)),
                        'recv_mb': self.safe_round(io.bytes_recv / (1024**2)),
                        'packets_sent': self.safe_int(io.packets_sent),
                        'packets_recv': self.safe_int(io.packets_recv)
                    }
                    total_sent += io.bytes_sent
                    total_recv += io.bytes_recv
            
            return {
                'timestamp': datetime.now().isoformat(),
                'interfaces': interface_data,
                'total_sent_mb': self.safe_round(total_sent / (1024**2)),
                'total_recv_mb': self.safe_round(total_recv / (1024**2)),
                'active_connections': len([c for c in connections if c.status == 'ESTABLISHED'])
            }
        except Exception as e:
            print("网络监控错误: " + str(e))
            return {'timestamp': datetime.now().isoformat(), 'interfaces': {}, 'active_connections': 0}
    
    def collect_all_stats(self):
        """收集所有性能数据"""
        stats = {}
        components = [
            ('cpu', self.get_cpu_stats),
            ('memory', self.get_memory_stats),
            ('disk', self.get_disk_stats),
            ('gpu', self.get_gpu_stats),
            ('network', self.get_network_stats)
        ]
        
        for name, func in components:
            try:
                data = func()
                if data:
                    stats[name] = data
            except Exception as e:
                print(str(name) + "组件错误: " + str(e))
        
        return stats
    
    def start_monitoring(self, interval=5, duration=60):
        """开始监控"""
        print("开始监控 - 间隔:" + str(interval) + "秒, 时长:" + str(duration) + "秒")
        
        self.monitoring = True
        start_time = time.time()
        sample_count = 0
        
        try:
            while self.monitoring and (time.time() - start_time) < duration:
                stats = self.collect_all_stats()
                
                if stats:
                    for component, data in stats.items():
                        self.data_buffer[component].append(data)
                    
                    self.print_status(stats)
                    sample_count += 1
                
                time.sleep(interval)
                
        except KeyboardInterrupt:
            print("\n用户中断监控")
            self.monitoring = False
        
        print("\n监控完成,样本数: " + str(sample_count))
        return self.data_buffer
    
    def print_status(self, stats):
        """打印实时状态"""
        time_str = datetime.now().strftime('%H:%M:%S')
        cpu_avg = stats.get('cpu', {}).get('cpu_avg', 0)
        mem_percent = stats.get('memory', {}).get('percent', 0)
        
        status_parts = ["时间:" + time_str, "CPU:" + str(cpu_avg) + "%", "内存:" + str(mem_percent) + "%"]
        
        # GPU状态
        gpu_info = stats.get('gpu', {})
        if gpu_info.get('gpu_count', 0) > 0:
            gpu_data = gpu_info['gpu_data'][0]
            gpu_util = gpu_data.get('gpu_util', 0)
            gpu_mem = gpu_data.get('mem_util', 0)
            status_parts.append("GPU:" + str(gpu_util) + "%(显存:" + str(gpu_mem) + "%)")
        
        # 磁盘状态
        disk_info = stats.get('disk', {})
        if disk_info and disk_info.get('usage'):
            first_disk = list(disk_info['usage'].values())[0]
            disk_usage = first_disk.get('percent', 0)
            disk_device = first_disk.get('device', 'Disk')
            status_parts.append(str(disk_device) + ":" + str(disk_usage) + "%")
        
        # 网络连接
        net_connections = stats.get('network', {}).get('active_connections', 0)
        if net_connections > 0:
            status_parts.append("连接:" + str(net_connections))
        
        print(" | ".join(status_parts))
    
    def analyze_performance(self):
        """性能分析"""
        if not self.data_buffer.get('cpu') or not self.data_buffer.get('memory'):
            return None
        
        cpu_values = [d['cpu_avg'] for d in self.data_buffer['cpu']]
        mem_values = [d['percent'] for d in self.data_buffer['memory']]
        
        avg_cpu = self.safe_round(sum(cpu_values) / len(cpu_values))
        avg_mem = self.safe_round(sum(mem_values) / len(mem_values))
        max_cpu = self.safe_round(max(cpu_values))
        max_mem = self.safe_round(max(mem_values))
        
        # GPU分析
        gpu_analysis = {}
        if self.data_buffer.get('gpu'):
            gpu_utils = [data['gpu_data'][0]['gpu_util'] for data in self.data_buffer['gpu'] 
                        if data.get('gpu_count', 0) > 0]
            if gpu_utils:
                gpu_analysis = {
                    'avg_util': self.safe_round(sum(gpu_utils) / len(gpu_utils)),
                    'max_util': self.safe_round(max(gpu_utils)),
                    'gpu_name': self.data_buffer['gpu'][0]['gpu_data'][0]['name']
                }
        
        # 磁盘分析
        disk_analysis = {}
        if self.data_buffer.get('disk'):
            disk_data = [data['usage'] for data in self.data_buffer['disk'] if data.get('usage')]
            if disk_data:
                first_disk_key = list(disk_data[0].keys())[0]
                disk_values = [data[first_disk_key]['percent'] for data in disk_data]
                disk_analysis = {
                    'avg_usage': self.safe_round(sum(disk_values) / len(disk_values)),
                    'max_usage': self.safe_round(max(disk_values)),
                    'disk_name': disk_data[0][first_disk_key]['device']
                }
        
        # 应用类型判断
        if avg_cpu > 80:
            app_type, suggestion = "计算密集型", "建议升级CPU或使用多线程优化"
        elif avg_mem > 80:
            app_type, suggestion = "内存密集型", "建议增加内存容量"
        elif gpu_analysis and gpu_analysis.get('avg_util', 0) > 70:
            app_type, suggestion = "GPU计算型", "GPU利用良好，可考虑并行计算"
        elif avg_cpu > 50 or avg_mem > 50:
            app_type, suggestion = "中等负载型", "硬件使用均衡"
        else:
            app_type, suggestion = "轻负载型", "硬件配置充足"
        
        return {
            'app_type': app_type,
            'suggestion': suggestion,
            'cpu_avg': avg_cpu, 'cpu_max': max_cpu,
            'mem_avg': avg_mem, 'mem_max': max_mem,
            'gpu_info': gpu_analysis,
            'disk_info': disk_analysis,
            'sample_count': len(cpu_values)
        }
    
    def generate_report(self):
        """生成分析报告"""
        analysis = self.analyze_performance()
        if not analysis:
            print("数据不足,无法分析")
            return
        
        vm = psutil.virtual_memory()
        system_info = {
            'os': platform.system() + " " + platform.release(),
            'cpu_cores': str(psutil.cpu_count()) + "逻辑/" + str(psutil.cpu_count(logical=False)) + "物理",
            'memory_gb': str(self.safe_round(vm.total / (1024**3)))
        }
        
        print("\n" + "="*60)
        print("硬件性能分析报告")
        print("="*60)
        print("系统: " + system_info['os'])
        print("CPU: " + system_info['cpu_cores'] + "核心")
        print("内存: " + system_info['memory_gb'] + "GB")
        
        if analysis['gpu_info']:
            gpu_info = analysis['gpu_info']
            print("GPU: " + gpu_info['gpu_name'])
            print("GPU平均利用率: " + str(gpu_info['avg_util']) + "%")
        
        if analysis['disk_info']:
            disk_info = analysis['disk_info']
            print("磁盘: " + disk_info['disk_name'] + " 平均使用率: " + str(disk_info['avg_usage']) + "%")
        
        print("\n性能分析:")
        print("应用类型: " + analysis['app_type'])
        print("CPU平均: " + str(analysis['cpu_avg']) + "% (峰值: " + str(analysis['cpu_max']) + "%)")
        print("内存平均: " + str(analysis['mem_avg']) + "% (峰值: " + str(analysis['mem_max']) + "%)")
        print("优化建议: " + analysis['suggestion'])
        print("样本数量: " + str(analysis['sample_count']))
        print("="*60)
        
        # 保存报告
        report_data = {
            'timestamp': datetime.now().isoformat(),
            'system_info': system_info,
            'analysis': analysis,
            'raw_data': dict(self.data_buffer)

        try:

        except Exception as e:
            print("保存报告错误: " + str(e))

### 附录：专业术语表

> **ALU（Arithmetic Logic Unit，算术逻辑单元）**：CPU内部负责执行算术和逻辑运算的硬件单元

> **Cache（缓存）**：位于处理器和内存之间的高速存储器，用于减少内存访问延迟

> **CUDA Core（CUDA核心）**：NVIDIA GPU中的基本计算单元，专门设计用于并行计算

> **Compute-Intensive（计算密集型）**：主要瓶颈在于计算能力而非内存或I/O的应用类型

> **FPU（Floating Point Unit，浮点运算单元）**：专门处理浮点数运算的硬件单元

> **GPU（Graphics Processing Unit，图形处理单元）**：专门设计用于并行计算的处理器

> **Memory-Intensive（内存密集型）**：主要瓶颈在于内存访问而非计算的应用类型

> **NUMA（Non-Uniform Memory Access，非统一内存访问）**：多处理器系统中内存访问时间因位置而异的架构

> **PCIe（Peripheral Component Interconnect Express）**：现代计算机中用于连接各种硬件组件的高速串行总线标准

> **SM（Streaming Multiprocessor，流式多处理器）**：NVIDIA GPU中的基本执行单元，包含多个CUDA Core

> **Stored Program Concept（存储程序概念）**：程序指令和数据存储在同一内存空间中的计算机架构理念

> **Throughput（吞吐量）**：单位时间内系统能够处理的任务数量

> **Von Neumann Architecture（冯·诺依曼架构）**：现代计算机的基础架构，包括处理器、内存、输入输出设备和总线

> **Warp（线程束）**：GPU中32个线程组成的执行单元，同时执行相同的指令
